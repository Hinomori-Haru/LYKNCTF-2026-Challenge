chall-3
=======

A flag checker. Feed it the flag; it prints "Correct!" if right, "Nope" if wrong.

Two builds of the SAME challenge are provided, pick the one for your OS:

  chall-3       Linux   x86_64 ELF
  chall-3.exe   Windows x86_64 PE

Usage:

  ./chall-3 'YOUR_FLAG_HERE'
      or
  ./chall-3
  flag: YOUR_FLAG_HERE

Flag format: LYKNCTF{...}
