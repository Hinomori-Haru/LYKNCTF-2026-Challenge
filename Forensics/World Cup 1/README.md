# WorldCup1 - Forensics Challenge

## Description
I woke up at 5 AM Vietnam time (UTC+7) to watch my favorite player Messi score goals, but instead I witnessed Cabo Verde playing so resiliently and bravely despite facing the defending champions for the full 120 minutes. The fairytale has ended, but the pride will remain forever.

## Challenge
Can you find the hidden flag in this image?

**Flag format:** `LYKNCTF{...}`

## Tools you might need
- exiftool
- steghide
- zsteg
- strings
- hexeditor

## Difficulty
Medium

## Files
- `worldcup1_challenge.png` - The challenge image

## Hints
- Multiple layers of steganography
- Check the metadata first
- The winning score might be important
- Look deeper into the pixels

Good luck!
